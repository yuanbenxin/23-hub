# 23-hub
23-hub是六安一中2024级23班的照片墙，网页源码由本新同学编写

**hub网址：https://23-hub.netlify.app/** [![Netlify Status](https://api.netlify.com/api/v1/badges/0ca15199-f6fc-4876-a3fc-9ec94e651c64/deploy-status)](https://app.netlify.com/projects/23-hub/deploys)


## 网页特色
* 平滑精致的外观--设计灵感来源于Win11 UI ，简约大气
* 图片尺寸自适应：根据图片尺寸自动调整卡片长宽高

## 23班照片墙行为准则

### 目的声明

本行为准则旨在为23班照片墙项目创建一个尊重、包容和安全的社区环境。我们相信，一个积极健康的社区氛围对于项目的成功至关重要。本准则适用于所有参与本项目的人，包括贡献者、维护者和用户，无论是在工作场所内还是在工作相关的社交活动中。

### 我们的价值观

- **尊重**：我们尊重每个人的尊严、隐私和独特性。在分享照片时，我们特别注意尊重他人的肖像权和个人空间。
- **包容**：我们欢迎来自不同背景的人参与，无论其年龄、性别、种族、宗教或任何其他个人特征。
- **责任**：我们对自己的行为负责，对他人负责，对社区负责。
- **透明**：我们在决策和沟通中保持开放和诚实。
- **协作**：我们鼓励建设性的反馈和合作，共同提升项目质量。

### 适用范围

本行为准则适用于：
- 项目仓库中的所有互动（包括GitHub issues、pull requests和代码审查）
- 与项目相关的任何会议、活动或社交场合
- 任何以项目名义进行的线上或线下交流

### 肖像权声明

本项目尊重每个人的肖像权。我们鼓励23班同学分享照片，但请确保您有权分享所提交的每张照片。如果您不是照片的拍摄者或照片中人物，请在提交前获得必要的许可。我们理解班级活动中的照片分享是集体记忆的一部分，但仍请尊重他人的隐私和意愿。

### 投诉报告流程

如果您遇到违反本行为准则的情况，我们鼓励您报告：

1. **直接沟通**：如果情况允许，您可以直接与相关人员沟通您的关切。
2. **联系维护者**：您可以向项目维护者（yuanbenxin）发送私信报告问题。
3. **正式投诉**：对于严重或持续的问题，请通过GitHub issue提交正式投诉，标记为"[CoC] 投诉"。

所有投诉将被认真对待，我们将尽快调查并在3个工作日内回复。我们承诺对投诉进行保密处理，并采取适当措施解决问题。

### 反骚扰政策

我们不容忍任何形式的骚扰，包括但不限于：
- 冒犯性的评论或图像
- 恐吓、跟踪或持续关注
- 未经同意的拍照或录音
- 干扰他人参与活动
- 任何其他令他人感到不适的行为

### 开源与许可

本项目以MIT许可证开源，详情请参阅[LICENSE文件](LICENSE)。yuanbenxin保留对本项目及其内容的最终管理权。虽然我们鼓励社区贡献，但所有提交的内容需符合本行为准则和项目目标。

### 责任与执行

项目维护者负责执行本行为准则。违反本准则可能导致：
- 临时或永久禁止参与讨论
- 拒绝或撤销pull request
- 从项目中移除

### 持续改进

我们将每六个月审查和更新本行为准则，以确保其有效性和相关性。我们欢迎社区成员对本准则提出改进建议。
